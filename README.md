
# Savaş Temalı Grafik Tasarım Projesi

Bu proje, Grafik Tasarım dersi kapsamında hazırlanmıştır.  
Çalışmada savaş teması; görsel tasarım, etkileşim ve ses desteğiyle dijital ortama aktarılmıştır.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Amaç
Bu projede amaç, grafik tasarım ilkelerini kullanarak tematik ve dikkat çekici bir dijital çalışma ortaya koymaktır.

## Çalıştırma
`index.html` dosyasını tarayıcıda açmanız yeterlidir.

## Not
Bu proje eğitim amacıyla hazırlanmıştır.
